# melongames

🎮 Interactive English learning games for kids

## Game 1: Melon Runner

Control a cute watermelon character with arrow keys to collect English words and avoid obstacles!

### How to Play
- Use **Arrow Keys** (⬆️⬇️⬅️➡️) to move
- Collect the English words to score points
- Avoid the obstacles
- Learn English while having fun!

### Tech Stack
- Next.js 14 (App Router)
- TypeScript
- CSS Modules
- Canvas API for game rendering
